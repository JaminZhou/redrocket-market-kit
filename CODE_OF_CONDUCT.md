# 行为准则

本项目欢迎围绕只读市场数据、Agent skill、CLI 可用性和开源维护质量的建设性贡献。

## 我们鼓励

- 尊重不同经验水平的使用者和贡献者。
- 用可复现的步骤、清晰的边界和事实证据讨论问题。
- 对金融、账户、安全相关内容保持谨慎，不公开粘贴敏感信息。
- 在不同意见中聚焦代码、文档、数据来源和风险控制，而不是个人。

## 不接受

- 人身攻击、骚扰、歧视、威胁或持续打扰。
- 在公开 issue、PR 或讨论中发布凭证、Cookie、验证码、账户标识、个人资产截图或身份信息。
- 诱导项目突破只读边界，例如增加交易、申购、赎回、转换、转账或确认能力。
- 把本项目输出包装成确定性投资建议、官方数据源或实时交易信号。

## 执行

维护者可以编辑、隐藏或删除违反本准则的内容，也可以临时或永久限制相关账号参与项目。

安全问题请按 [SECURITY.md](SECURITY.md) 处理，不要在公开 issue 中披露敏感细节。
